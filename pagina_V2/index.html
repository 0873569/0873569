Samenvatting

Bootstrap is een open library gebasseerd op HTML, CSS en jQuery. Het wordt wereldwijd gebruikt door designers, front-end developers en bedrijven (Snapchat, Spotify etc). De open library bevat goede documentatie, waardoor je met een lage programmeerkennis een volledig website kan bouwen. Het wordt vaak gebruikt als basis layout en vervolgens gecustomized of gebruikt voor een prototype die in korte periode gerealiseerd moet worden.

Bootstrap bevat een gridsysteem om het responsive te maken, CSS opmaak voor tekst en daarnaast diverse elementen zoals sliders, cards, responsive images, shadows en uitlijningen.

Bootstrap is een open libary gebasseerd op HTML, CSS en jQuery. De library integreer je in je eigen html document door middel van metatags: stylesheets en (java)scripts. Daarnaast biedt Bootstrap gratis voor geschreven code voor diverse content. Je kunt hierdoor in korte tijd een basis layout (of prototype) maken voor een website of app. Denk hierbij aan: Tekst opmaak, responsive images, responsive grid systeem, pop ups, cards, navigatie menu etc.
Je kunt onder het kopje functionaliteiten meer info vinden.

Bootstrap wordt internationaal gebruikt door duizenden mensen. Dit zijn mensen die werkzaam zijn in de IT branch, maar ook mensen die zichzelf HTML willen aanleren of een website bouwen. Een kleine greep door wie Bootstrap gebruikt wordt: front-end developers, UI/UX Designers en bedrijven (Snapchat, Spotify, AD.nl, Telegraaf, NOS).

Bootstrap is handig, omdat je met een vrij lage programmeerkennis en gehele (prototype) website in elkaar kan zetten. Bootstrap heeft alle classes voor opmaak en vormgeving al voor je aangemaakt. Denk hierbij aan Headings, paragraaftekst, url's, maar ook cards, tooltips etc.

De HTML code die je nodig hebt voor bovenstaande items, kun je vinden in de documentatie op de website van bootstrap.
Doordat je alle code kunt vinden en mag kopieren van de website van bootstrap, kun je in een korte periode een website bouwen. Daarnaast is deze website direct responsive voor mobiel, tablet en laptop. Deze open library bespaart veel tijd in vergelijking met zelf schrijven!

Je kunt de opmaak van de elementen eventueel zelf aanpassen, hier is wel basis HTML en CSS kennis voor vereist.

Bootstrap gebruik je door de metatags, CSS en scripts te integreren in je eigen HTML document of je download het voorbeeld document van bootstrap.
In dit document zit alles wat je nodig heb om bootstrap te laten werken: Het grid systeem, de opmaak voor tekstelementen en de opmaak voor alle componenten en utilities die je standaard al vanuit bootstrap kan gebruiken.

Functionaliteiten

Layout

Bootstrap staat bekend om het grid systeem dat zij hebben ontwikkeld. Dit grid systeem is de basis voor een responsive website. Het is gebasseerd op 1 rij met hierin 12 kolommen die naast of onder elkaar worden geplaatst in de website. Of de kolommen naast of onder elkaar worden geplaatst is afhankelijk van de resolutie van het device. De rij met 12 kolommen kun je zelf indelen zoals je wilt. De rij moet alleen in totaal 12 kolommen bevatten. Zie het voorbeeld hieronder.

De kolommen kun je verdelen in groepen. Dit wordt gedaan door classes die je gebruikt binnen de html. Je kunt in 1 rij bijvoorbeeld 2 groepen maken. Deze 2 groepen moeten bij elkaar 12 kolommen bevatten. Het is vrij in te delen: 6 -6, 5-7, 4-8, 1-11 etc.

Bootstrap gebruikt 'breakpoints' om te bepalen wanneer kolommen naast elkaar of onder elkaar worden geplaatst. Deze breakpoints zijn resolutie afhankelijk. Een aantal van deze standaard breakpoint zijn:
• max-width: 576px
• max-width: 768px
• max-width: 992px
• max-width: 1200px
• min-width: 1200px

Hierbij is uit gegaan van de 'standaard' resolutie van devices: mobiel, tablet, Laptop met HiDPI, laptop met MDPI en full HD.

Content

Bootstrap heeft bijzonder veel voorgeschreven classes voor content en componenten. Hierdoor zijn veel standaard HTML elementen al opgemaakt. Denk hierbij aan de opmaak van tekst titels en paragrafen: type font, font grootte, line-height etc. Deze opmaak kan simpel en snel worden aangeroepen door een zin te coderen met <h1> voorbeeld H1 </h1>. Elke H class heeft zijn eigen lettergrootte. Deze style regels zijn makkelijk te overschrijven door in je eigen CSS een class te maken met .H1 en je eigen style regels.

Behalve de opmaak voor Headers heeft bootstrap dit ook voor veel andere tekst elementen gedaan: Bold, italic of highlight tekst. Deze classes kun je wederom overschrijven met je eigen style regels, maar dit is niet aan te raden. De opmaak van deze elementen staat los van het type font, font grootte en lineheight. Je gebruikt deze elementen dan ook vaak in combinatie met een van de headers of paragraaf elementen, waarbij je al je eigen opmaak mee geeft.

Responsive image

Voor Images heeft bootstrap beperkte code gepubliceerd, maar heel veel meer is er ook niet nodig. Ze hebben 1 regel gepubliceerd voor een reponsive image. Deze zorgt in de breedte dat hij altijd op 100% is van de ruimte.
Let hierbij op: Wanneer jij een reponsive image plakt in een kolom van 4 breed, dan zal hij alleen de ruimte van de kolommen pakken en niet van je hele pagina.
